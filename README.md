# Humbucker Pickup Analyzer

A professional web-based tool for guitar technicians and luthiers to analyze humbucker pickup wire color codes and phase relationships.

## Features

- **Comprehensive Manufacturer Database**: Supports 36+ pickup manufacturers with their specific wire color codes
- **Phase Conflict Detection**: Automatically identifies potential phase issues when combining pickups
- **Custom Pickup Support**: Add your own pickup specifications
- **Print-Friendly**: Generate professional documentation for customer records
- **Save/Load Configurations**: Export and import your pickup setups as JSON files

## Live Demo

[View Live Tool](https://vanlaarhovenguitars.github.io/humbucker-pickup-analyzer/)

## Usage

Simply open `index.html` in a web browser. No server or installation required.

1. Select pickups from the manufacturer database
2. Assign them to guitar positions (bridge, middle, neck)
3. Review phase compatibility warnings
4. Print or save your configuration

## For Guitar Technicians

This tool uses the practical "pull off rapidly" multimeter testing technique for phase verification. All manufacturer data has been verified against actual pickup specifications.
